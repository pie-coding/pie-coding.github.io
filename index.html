Intro to Programming.

Overview

  • 2 courses to teach
  • very basic, simple class to teach the fundamentals
  • relaxed and casual attitude towards coding
  • learn the basics
  • use MIT's Scratch to build an application
  • use Java to build an application
  • 2 hours per class
  • hands on mentoring
  • 2-3 lessons per class

All software will be provided offline (USB drives) so no internet connection should be required. We will only be using free and open source software to ensure equal access for tools for everyone.

First Course

  • Learn to program using MIT's Scratch
  • Teach users to create their own programs
  • Events: click on it to to have it react
  • Operators and Math: Blackjack numbers game
  • Logic: Conditional game
  • Finally, will discuss remixing for students to further study

Second Course

  • Teach Java (IDE to be decided)
  • Teach the basics of Android programming
  • Create new applications (similar to taught last time)
  • Hello, world application
  • Create a Tic Tac Toe game
  • Create basic calculator
